# Fibonacci-Series-using-iteration
Fibonacci Series using Iteration

This Python program prints the Fibonacci series using a for loop (iteration).

What is Fibonacci Series?

A Fibonacci series is a sequence where:

The first number is 0
The second number is 1
Every next number is the sum of the previous two numbers

Example:

0 1 1 2 3 5 8 13 ...
How the Program Works
User enters how many terms they want.
Variables:
a = 0
b = 1
Loop runs n times.
Each time:
Print a

Find next number using:

c=a+b

Update values:
a = b
b = c
Example Output
Enter the number of terms you want in the Fibonacci series : 7
Fibonacci series is : 0 1 1 2 3 5 8
Time Complexity
O(n) → because loop runs n times
Space Complexity
O(1) → only a few variables are used
